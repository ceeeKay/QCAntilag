# QCAntilag

Antilag for QuakeC

A "very poor man's implementation" of lag compensation, implemented with zero
client engine changes- thus compatible with any QuakeC client/server.

Known to work/tested with:

* Quake Enhanced (2021 rerelease)
* FTE Server
